# week1_정리
```html
  <h1>HI!</h1>
  <h2>HI!</h2>
  <h3>HI!</h3>
  <h4>HI!</h4>
  <h5>HI!</h5>
  <h6>HI!</h6>
```
6단계로 제목을 나타낸다 (h1이 가장 큼)
```html
  <p>HI!</p>
```
하나의 문단을 나타낸다
```html
  <a href="https://github.com/" target="_blank">깃허브</a>
```
href특성을 통해 URL연결 하이퍼링크를 만든다
```html
  <hr>
  <b>
  <u>
  <i>
  <br>
```
위에서 부터 차례로 가로줄, 굵은글씨, 밑줄, 기울기, 줄바꿈
